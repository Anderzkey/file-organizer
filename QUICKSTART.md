# File Organizer - Quick Start

## What It Does
File Organizer is a chat-based app that organizes your files using natural language commands.

## How to Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the app**
   ```bash
   npm run dev
   ```

3. **Open in browser**
   Go to: `http://localhost:3000`

## How to Use

Type commands in the chat box like:
- `"List my downloads"`
- `"Organize downloads by file type"`
- `"Move all PDFs to documents"`
- `"Show me large files"`

Just describe what you want, and the AI will handle it.

## What Happens
1. You type a command
2. AI understands what you want
3. It executes the file operation
4. You see the results in the chat

That's it! Simple and natural.
