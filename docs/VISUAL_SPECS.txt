  Padding:         px-6 = 24px, py-2 = 8px

  Hover:           #1d4ed8 (hover:bg-blue-600)

================================================================================
SPACING SYSTEM (BASED ON TAILWIND)
================================================================================

Base Unit: 4px (1 Tailwind unit)

Multiples Used:
  1px:  borders (not Tailwind unit, CSS border-width)
  2px:  focus ring width (focus-visible:ring-2)
  4px:  mt-1 (spacing between input and dropdown)
  6px:  (mr-2 for icon spacing = 8px in Tailwind)
  8px:  px-0.5 (not used), py-2 vertical component
  12px: py-3 (mobile touch targets)
  16px: px-4 (horizontal padding standard)
  24px: px-6 (buttons, wider components)

Applied in Dropdown:
  mt-1:     4px margin-top (input to dropdown gap)
  px-4:     16px padding-left and right (container and items)
  py-2:     8px padding-top and bottom (items, desktop)
  py-3:     12px padding-top and bottom (items, mobile)
  mr-2:     8px margin-right (icon to text spacing)
  ml-6:     24px margin-left (command text indent)
  gap-2:    8px gap between input and button
  rounded-lg: 8px border-radius

INTERACTIVE STATES - VISUAL PROGRESSION
================================================================================

STATE 1: AT REST (Dropdown Closed)
  Input:
    Border:        #e5e7eb (gray-200), 1px
    Background:    white
    Text:          #374151 (gray-700)
    Cursor:        text
    Appearance:    Normal input field

STATE 2: INPUT FOCUSED (Dropdown Opens)
  Input:
    Border:        #3b82f6 (blue-500), 1px
    Background:    white
    Cursor:        text
  Dropdown:
    Appears:       Below input, 4px gap
    Items:         All white background, gray text
    Opacity:       100% (fully visible)
    Interactable:  Yes

STATE 3: ITEM HOVERED (Desktop)
  Hovered Item:
    Background:    #eff6ff (blue-50)
    Text:          #2563eb (blue-600)
    Font Weight:   600 (semibold) - ENHANCED
    Cursor:        pointer
  Other Items:
    Unchanged (white background, gray text)
  Input:
    Border:        Still blue (focused)

STATE 4: ITEM KEYBOARD FOCUSED
  Focused Item:
    Background:    #eff6ff (blue-50)
    Text:          #2563eb (blue-600)
    Font Weight:   600 (semibold)
    Ring:          2px #3b82f6 ring around item
    Cursor:        default
  Other Items:
    Unchanged
  Input:
    Border:        Still blue

STATE 5: ITEM CLICKED
  Action:
    Item Selected  ✓
    Input Filled   ✓ (populates with command)
    Dropdown Closed ✓ (disappears)
    Form Submitted ✓ (auto-submit after state update)
  Result:
    Dropdown:      Hidden (z-index: -1 or display: none)
    Input:         Blurred after auto-submit
    Focus:         Returned to input
    Chat:          Message appears (async)

STATE 6: INPUT BLURRED (Dropdown Closes)
  Dropdown:
    Disappears:    Immediately
    Animation:     None (instant)
  Input:
    Border:        Back to #e5e7eb (gray-200)
    State:         Ready for next input

STATE 7: DISABLED (While Loading)
  Input:
    Opacity:       50% or grayed out
    Background:    #f3f4f6 (gray-100)
    Cursor:        not-allowed
    Interactions:  Disabled
  Dropdown:
    Options:       Hidden or grayed out (implementation choice)
    Interactable:  No

================================================================================
KEYBOARD NAVIGATION SPECIFICATION
================================================================================

When Dropdown is CLOSED:
  ArrowDown:       Open dropdown, focus first item
  ArrowUp:         Open dropdown, focus last item
  Type:            Add to input, keep dropdown closed

When Dropdown is OPEN:
  ArrowDown:       Move focus down to next item (wrap to first)
  ArrowUp:         Move focus up to previous item (wrap to last)
  Enter:           Select focused item, close dropdown, submit form
  Escape:          Close dropdown, return focus to input
  Tab:             Move to next focusable element (out of dropdown)
  Shift+Tab:       Move to previous focusable element

Focus Ring Appearance:
  Ring:            2px solid #3b82f6 (blue-500)
  Offset:          1px from element
  Visibility:      Always visible
  Contrast:        8.6:1 against white (WCAG AAA)

Focus Management:
  Initial Focus:   Input field (tab in from previous element)
  Dropdown Open:   Focus moves to first item (ArrowDown or manual)
  After Selection: Focus returns to input
  Click Outside:   Dropdown closes, focus remains where it was
